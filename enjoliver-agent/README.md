# Enjoliver Agent


The enjoliver agent is a daemon running on each CoreOS container linux node of the Kubernetes cluster.

The purpose of this daemon is provide local information.


**/healthz**
* vault etcd
* fleet etcd
* kube etcd
* vault
* kubelet
* kube apiserver
* rkt api


**/version**
* etcd
* rkt
* hyperkube
* iproute2
* kernel
* fleet
* vault
